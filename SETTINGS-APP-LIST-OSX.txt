----------------------------------
SETTINGS - APP LIST (OSX)
----------------------------------

This list provides essential applications for navigation and carrying out tasks.
Dock has the ideology of grouping similar types of apps together.
However, some core applications are hidden as they are brought up via keyboard shortcuts (see SETTINGS-KEYBOARD-MAPPINGS.txt).


----------------------------------
APPS
----------------------------------
Navigation
- Better Touch Tool (supports Touch Bar customization on Macbook Pro; necessary to have normal mouse scrolling while using natural scrolling on trackpad)
- Hyper Dock        (useful for window snapping)
- Logitech G Hub    (supports mouse macros)
- Smartset          (Kinesis split ergonomic keyboard tool; macros: see SETTINGS-KEYBOARD-MACROS.txt)
-----------------
Taskbar/Menu
- Bartender         (Menu Bar customization allowing for at-a-glance performance stats and hiding always-open background services like BTT)
- iStat Menus       (easy-access performance stats via Notification Center)
- itsycal           (bigger calendar pulldown from Menu Bar than what is provided out of box by Osx)
-----------------
Other
- Atom
- Chrome            (with Bookolio and Adblock plugins)
- Foxit Reader      (cross-platofrm PDF reader with customizable background color which eases screens on eyes)
- Inkscape          (open-source Adobe Illustrator alternative for Osx)
- iTerm2            (terminal for Osx)
- LibreOffice       (cross-platform free office suite)
- Microsoft Remote Desktop
- Slack
- Spotify
- Sublime Text
- The Unarchiver    (compression tool supporting a plethora of formats)
- Thunderbird       (allows for multiple email accounts to be used in one place, password-protected, and with any server authentication)
- VSCode
- Zoom/Other
-----------------
